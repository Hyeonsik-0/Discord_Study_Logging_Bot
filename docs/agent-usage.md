# AI Agent 활용 전략

## Tool
Claude Code (Anthropic)

## 활용 범위
- 아키텍처 설계 검토 및 개선 제안
- 기능 단위 코드 생성 및 구현 가이드 제공
- 디버깅 및 오류 원인 분석
- 코드 리팩토링 및 구조 개선
- 기술 선택에 대한 트레이드오프 분석

## 활용 원칙

본 프로젝트에서는 Claude를 단순 코드 생성 도구가 아닌, 설계·구현·검토를 함께 수행하는 협업 파트너로 활용한다.

단, **전체 시스템 구현을 Agent에 의존하지 않고**, 개발자의 이해를 기반으로 점진적으로 적용한다.

## 운영 가이드라인

`CLAUDE.md` 에 정의된 행동 가이드라인에 따라 다음 원칙을 적용한다.

1. **Think Before Coding** — 가정 명시, 트레이드오프 표면화
2. **Simplicity First** — 요구된 최소한만 구현, 추측 기반 추상화 금지
3. **Surgical Changes** — 요청된 부분만 수정, 무관한 코드는 건드리지 않음
4. **Goal-Driven Execution** — 검증 가능한 성공 기준 정의 후 반복
